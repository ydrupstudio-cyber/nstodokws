# NS_To-Do v4

인하대병원 신경외과 의국용 공유 할일 웹앱.

## v4 신규 기능

- 🔔 웹 푸시 알림 (긴급 추가/변경 + 시간 임박)
- 📅 미래 날짜에 할일 추가
- ⏭ 다가오는 일정 14일 미리보기
- 📌 완료 섹션 접기/펼치기
- 💬 댓글 기능
- 🏥 병동 칩 (9층, SB, EB, MA, MB, CCU)
- ⇅ 정렬 옵션 (시간/병동/교수/담당자)
- ⭐ 자주 쓰는 할일 템플릿
- 👨‍⚕️ 교수님 이니셜 시스템
- 🔍 대소문자 통일 검색 + 교수님별 필터
- 📊 통계 화면 (주간/월간)

## 적용 순서

1. **SQL 실행**: `SUPABASE_SQL_V4.sql` Supabase SQL Editor에서 실행
2. **VAPID 키 생성**: 푸시 알림용 (안내문 참조)
3. **Vercel 환경 변수 추가**:
   - `NEXT_PUBLIC_VAPID_PUBLIC_KEY`
   - `VAPID_PRIVATE_KEY`
4. **GitHub 코드 교체** 후 자동 배포 대기
5. **본인 폰에서 테스트**
